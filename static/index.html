Capteurs / sorties sur cet ESP (firmware)

                    
Choisis dans la liste ce qui est câblé sur l'ESP pour ne pas le perdre au flash. Ces entités restent visibles par HA et Athena en parallèle.

                    

                    
                    

                        
                        
                        
                        LED
                    

                    
GPIO48 = LED RGB intégrée (DevKitC-S3, rien à câbler). Bandeau externe WS2812B → broche DATA sur une GPIO libre (ex. GPIO38) + indique le nombre de LED. >1 LED → effets adressables par état (défilement, arc-en-ciel, scintillement). Le schéma de câblage est inclus en tête du YAML généré.

⚠️ Proxmox en IP privée : ajoute son hôte à NET_GUARD_ALLOW_HOSTS (onglet Nextcloud) pour l'anti-SSRF. Les actions (démarrer/arrêter) sont toujours soumises à validation.

                


                

                    
👁️ Vigie — surveillance proactive

                    
Athena réagit à des événements POUSSÉS par tes outils de supervision (Zabbix, Grafana, LibreNMS, Home Assistant, traps SNMP via forwarder…). Rien ne tourne en boucle : le Vigie ne se réveille que sur événement, analyse, t'alerte (et propose un correctif, validé via Telegram pour les actions sensibles).

                    

                        
                    

                    

                        
                        
                    

                    

                        
                    

                    

                        
                        
                    

                    

                        
                        
                    

                    

                        
                        

                            
                            
                        

                        
Tes sources poussent : POST /api/events (en-tête X-Event-Token) avec {type, source, severity, message}.
