Bienvenue sur AthenaDesign Studio !

Je suis votre assistant de design. Indiquez-moi ce que vous souhaitez concevoir dans le champ ci-dessous. Vous pouvez créer :

  • Des interfaces web (HTML/CSS/JS) comme des tableaux de bord, formulaires ou pages vitrines.
  • Des scripts Python interactifs pour tracer des graphiques scientifiques (Matplotlib/Plotly) ou manipuler des données.
  • Des présentations PowerPoint (.pptx) épurées et professionnelles générées par script Python.

Astuce : Sans clé API configurée, l'application fonctionne en mode hors-ligne avec des démos interactives instantanées (essayez de taper "dashboard", "plot" ou "powerpoint" !).
